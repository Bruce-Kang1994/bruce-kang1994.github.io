# kangxin.me Aesthetic Fence — "Ink & Paper"

## Design Philosophy
Restrained editorial authority. Like a well-designed independent research journal — typography IS the design, not decoration. Confident without being cold. Structured without being rigid. The content is the product; the design serves reading.

**Personality**: A senior practitioner who publishes their workbook — shows work, not credentials.
**Reference sites**: Craig Mod (craigmod.com), Matter journal, Stripe Press

## Typography
- MUST use Source Serif 4 for all headings (H1-H3)
- MUST use Source Sans 3 for all body text, labels, and UI elements
- MUST use JetBrains Mono ONLY for code blocks and inline code
- MUST limit to 5 font size levels (H1, H2, H3, Body, Caption)
- MUST use text-wrap: balance for headings
- MUST use text-wrap: pretty for body paragraphs
- SHOULD set Chinese fallback stack: 'PingFang SC', 'Noto Sans SC', 'Microsoft YaHei', sans-serif

## Colors
- MUST use #FAF9F7 (warm paper) as light mode page background
- MUST use #0C0A09 (warm near-black) as dark mode page background
- MUST use #9A3412 (deep burnt sienna) as primary accent in light mode
- MUST use #C2410C as primary accent in dark mode
- MUST maintain 4.5:1 contrast ratio for all text
- MUST add warm undertone to all neutral grays (stone scale, never pure gray)

## Section Labels
- MUST use Source Sans 3 (not monospace) for section labels
- MUST set labels as uppercase, 11px, weight 600, letter-spacing 0.1em
- SHOULD use --text-3 color for labels

## Spacing
- MUST use 8px base grid for all spacing
- MUST vary section spacing: 64px (tight) / 96px (standard) / 128px (generous)
- NEVER use uniform spacing between all sections

## Motion
- MUST use CSS-only animations (no GSAP, no JS animation libraries)
- MUST use Scroll Reveal: translateY(12px) + opacity, 400ms ease-out
- MUST respect prefers-reduced-motion: reduce
- SHOULD use subtle hover underline-offset transitions for links
- NEVER use bounce or elastic easing
- NEVER animate layout properties (width, height, margin, padding)

## NEVER
- NEVER use Inter, Roboto, or system sans-serif as heading font
- NEVER use blue (#2563eb or similar) as accent color
- NEVER use purple-to-blue gradient
- NEVER use JetBrains Mono for section labels (that's Clean Tech, not Editorial)
- NEVER use paper grain texture or pseudo-element decorations
- NEVER use drop caps
- NEVER use glassmorphism or glass-effect backgrounds
- NEVER use card-inside-card layout
- NEVER use 6+ equal-sized feature cards in a grid (AI slop signature)
- NEVER use gradient text

## Reference
- craigmod.com — editorial restraint, typography as design
- stripepress.com — warm neutrals, serif headings, generous whitespace
